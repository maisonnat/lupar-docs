# Guías Paso a Paso

Esta sección contiene guías prácticas para las tareas más comunes en Lupar. Cada guía está diseñada para que puedas completarla en minutos, con instrucciones claras y capturas de pantalla de referencia.

## Guías de Evaluación

- **[[how-to-assess-a-tool]]** – Cómo evaluar el compliance de una herramienta de IA contra las regulaciones aplicables
- **[[how-to-authorize-a-tool]]** – Cómo autorizar una herramienta para uso oficial en la organización
- **[[how-to-dismiss-false-positive]]** – Cómo descartar detecciones que son falsos positivos

## Guías de Reportes

- **[[how-to-generate-compliance-report]]** – Cómo generar un reporte de compliance HTML autocontenido
- **[[how-to-prepare-for-audit]]** – Cómo preparar tu organización para una auditoría de IA

## Guías de Configuración

- **[[how-to-configure-organization]]** – Cómo configurar los datos de tu organización para reports
- **[[how-to-backup-restore]]** – Cómo exportar e importar datos de Lupar

## Flujo de Trabajo Recomendado

Para organizaciones que recién comienzan con Lupar, recomendamos este orden:

1. Configurá los datos de tu organización en [[settings]] → ver [[how-to-configure-organization]]
2. Revisá las herramientas detectadas en el [[inventory]]
3. Evaluá cada herramienta → ver [[how-to-assess-a-tool]]
4. Autorizá las herramientas aprobadas → ver [[how-to-authorize-a-tool]]
5. Descartá los falsos positivos → ver [[how-to-dismiss-false-positive]]
6. Generá tu primer reporte → ver [[how-to-generate-compliance-report]]
7. Preparate para auditorías → ver [[how-to-prepare-for-audit]]

> [!tip] Si estás evaluando Lupar por primera vez, empezá por [[how-to-assess-a-tool]] para entender el flujo de evaluación. Es la actividad central del sistema.
